The application uses a GUI and is written in C# .NET using WPF (Windows Presentation Foundation).

The application uses an MVC design so the code is separated according to function (Model, View, or Controller)

The main files with actual code content are:

Simulator/PCB.cs				(Model)
Simulator/ProcessStatus.cs		(Model)
Simulator/SimulatorModel.cs		(Model)
Simulator/SimulatorViewModel.cs	(Controller)
Simulator/MainWindow.xaml		(View)
